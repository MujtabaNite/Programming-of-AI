Welcome to the official repository for my **Programming of AI** module (University Semester 4). This repository serves as a comprehensive log of laboratory work, practical assignments, and datasets used to explore the foundations of Artificial Intelligence and Machine Learning.

## 📁 Repository Structure

The project is organized into two main categories:

### 1. 📊 Datasets
A collection of curated CSV files used for training models and practicing data manipulation:
- **`creditcard.csv`**: For fraud detection and unbalanced data practice.
- **`depression_dataset.csv`**: Real-world data for sentiment and classification tasks.
- **`titanic_toy.csv`**: Classic dataset for binary classification.
- **`Handling_missing.csv`**: Specialized for imputation techniques.
- ... and more.

### 2. 🧪 Lab Work
Practical implementations organized by instructor, covering end-to-end ML workflows:

#### 🔹 Key Topics Covered:
- **Data Preprocessing:** Handling missing values (Mean/Median/Mode Imputation, KNNImputer, Iterative Imputer).
- **Feature Engineering:** Label Encoding, Outlier detection and removal.
- **Dimensionality Reduction:** Principal Component Analysis (PCA).
- **Model Training:** K-Nearest Neighbors (KNN), K-Fold Cross-Validation.
- **Data Visualization:** Exploratory Data Analysis (EDA) using Matplotlib and Seaborn.

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** NumPy, Pandas, Scikit-Learn, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook / Anaconda

## 🚀 How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/MujtabaNite/Programming-of-AI.git
   ```
2. Install dependencies:
   ```bash
   pip install pandas numpy scikit-learn matplotlib seaborn
   ```
3. Open any `.ipynb` file in Jupyter Lab or VS Code to explore the implementations.
